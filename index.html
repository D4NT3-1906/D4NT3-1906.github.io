First Time Setup

The very first Admin account cannot be created from this page — it must be created directly in Firebase Console.
Go to Firebase Console → Authentication → Add user and create a user with email yourname@attendiq.local and your chosen passcode.
Then go to Realtime Database and manually add the entry:
users → {uid from Auth} → { name:"YourName", role:"admin", subjectIds:[] }
After that, all future users are created from this panel.
