Admin Panel

Manage subjects and user credentials

Add New Subject

The Subject ID must exactly match the SUBJECT_ID constant in the ESP32 firmware for that device.

Add / Change User Credentials

Users are defined in the USERS object at the top of index.html. To add a teacher or change a passcode, open the file and edit that block:

"mr_rahul": { displayName: "Mr. Rahul Das", passcode: "newpassword", role: "teacher", subjects: ["cs301", "cs303"] }

Then commit & push to GitHub — the dashboard updates automatically via GitHub Pages.

ESP32 Device Notes

Each ESP32 is flashed with one SUBJECT_ID. To run multiple subjects at the same time, use multiple ESP32 boards, each with a different Subject ID. The Firebase Database Secret for the ESP32 is at: Firebase Console → Project Settings → Service accounts → Database secrets.
